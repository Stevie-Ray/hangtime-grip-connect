# CTS500

[CTS500](https://www.alibaba.com/product-detail/Mini-Climbing-Training-Scale-CTS500-Aluminum_1601637814595.html) is
listed as a climbing training scale and marketed as "Jlyscales CTS500" by supplier Hunan Jinlian Cloud Information
Technology Co., Ltd. The supplier website is [huaying-scales.com](https://www.huaying-scales.com/).

## Basic usage

```ts
import { CTS500 } from "@hangtime/grip-connect"

const device = new CTS500()
device.notify((data) => console.log(data.current))

await device.connect(
  async () => {
    console.log("Model:", await device.model())
    console.log("Serial:", await device.serial())
    device.disconnect()
  },
  (err) => console.error(err),
)
```

## Methods

CTS500 supports all [shared methods](/devices/#shared-methods) (connect, disconnect, isConnected, notify, active, read,
write, download). See [Device interface](/api/device-interface) for details.

### Device-specific

| Method           | Returns                        | Description        |
| ---------------- | ------------------------------ | ------------------ |
| `firmware()`     | `Promise<string \| undefined>` | Firmware version.  |
| `hardware()`     | `Promise<string \| undefined>` | Hardware version.  |
| `manufacturer()` | `Promise<string \| undefined>` | Manufacturer info. |
| `model()`        | `Promise<string \| undefined>` | Model number.      |
| `serial()`       | `Promise<string \| undefined>` | Serial number.     |
| `software()`     | `Promise<string \| undefined>` | Software version.  |

## Notes

- The current parser accepts ASCII packets when present and otherwise assumes a 16-bit centi-kilogram payload on `FFE1`,
  based on similar scale integrations in the library.
- If your unit reports incorrect values, capture a sample notification payload and update the parser in
  `packages/core/src/models/device/cts500.model.ts`.

See [Device interface](/api/device-interface) and [Guide](/guide) for connection and streaming.
